# 上傳 Image

專案開啟後，SaaS應用開發者便可透過docker push的指令將鏡像上傳至AM的專案內。

# 指令格式

  • Tag
docker tag SOURCE_IMAGE[:TAG] harbor.corepro.com.tw/{{**your_App_Name**}}/IMAGE[:TAG]
  • Push
docker push harbor.corepro.com.tw/{{**your_App_Name**}}/IMAGE[:TAG]

注意

若有網頁需求，須將路徑改成相對路徑。

<link href=‘/css/style.css’ >
改成
<link href=‘./css/style.css’> 或 <link href=‘css/style.css’>
